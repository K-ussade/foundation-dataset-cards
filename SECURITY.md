# Security Policy

## Reporting Sensitive Issues

Do not open a public issue containing secrets, raw data, private datasets, donor records, student data, school private data, private telemetry, exact sensitive infrastructure locations, or sealed YOSO-YAi LLC IP.

Use a private maintainer contact path until a formal Foundation security contact is published.

## Public Issue Scope

Public issues may discuss:

- broken links;
- dataset card wording;
- privacy exclusion improvements;
- provenance standard improvements;
- maintenance standard improvements;
- graph rendering issues;
- public boundary improvements;
- status-language corrections.

Public issues must not include:

- credentials, tokens, or API keys;
- raw datasets;
- private data;
- PII examples;
- private review records;
- donor, volunteer, student, school, customer, or private partner data;
- private telemetry;
- security-sensitive NEURONA operational information.

## Supported Scope

This repository is documentation-only. It does not collect, host, sanitize, publish, or release datasets.

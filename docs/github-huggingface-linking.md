# GitHub To Hugging Face Linking

## Purpose

This document defines how GitHub companion documentation and Hugging Face dataset cards should link to each other.

## Linking Rules

| Link | Rule |
| --- | --- |
| GitHub to Hugging Face | Link only after the Hugging Face dataset repository exists and status is reviewed. |
| Hugging Face to GitHub | Link to the companion README, sanitation checklist, governance review status, release notes, and maintenance plan. |
| `218.network` | Use a context placeholder until the reviewed public page exists. |
| Roadmap links | Planned items may link to roadmap status without claiming release. |

## Non-Claim Rule

A placeholder link is not evidence of release. Use `planned` until the public artifact exists with reviewed card content.

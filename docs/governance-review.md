# Governance Review

## Purpose

This document defines the governance review required before a dataset card or dataset release can be described as released.

## Required Gates

| Gate | Required outcome |
| --- | --- |
| Source authority | Source rights, licensing posture, and provenance are reviewed. |
| Privacy authority | PII, sensitive data, and private records are excluded or approved for public release. |
| Program authority | Foundation program context is public-safe. |
| Security authority | Sensitive infrastructure and NEURONA operational details are excluded. |
| Maintenance authority | Correction, takedown, update, and deprecation path exists. |
| Publication authority | Human governance review approves release status. |

## Status Rule

Use `planned` until the public Hugging Face dataset repository, dataset card, release notes, governance review, and maintenance plan exist.

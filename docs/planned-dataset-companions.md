# Planned Dataset Companions

## Purpose

This registry lists planned Hugging Face dataset names and the dataset-card release documentation required before any dataset can move beyond planned status.

## Registry Rules

- Add planned datasets here before creating dataset-specific cards.
- Keep status planned until public release documentation exists.
- Do not create a Hugging Face dataset repo just to reserve a name unless it is clearly scaffolded with no data and no release claim.
- Do not publish dataset-specific cards with fabricated collection, sanitation, PII, licensing, maintenance, or release data.
